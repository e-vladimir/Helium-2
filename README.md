# PET-Проект для ветки развития в области мобильной разработки.

## Предпосылки проекта
* Стоковая галерея медленная
* Стоковая галерея перегружена ненужными функциями

## Цель проекта
1. Наступить на всевозможные грабли мобильной разработки
2. Наступить на всевозможные грабли работы с медиа
3. Наступить на всевозможные грабли работы с файлами
4. Сделать приятный интерфейс
5. Добиться быстрой работы приложения
